# Sample reports（對外展示）

本目錄可提交**去識別化**範例報告，供 clone 後無需跑 sync 即可了解輸出格式。

日常產物寫入 **`reports/daily/`** 或 **`reports/research/`**（見 [reports/README.md](../README.md)）；僅 `reports/samples/` 納入版控。

### Canonical vs scratch

| 類型 | 路徑 | 範例 |
|------|------|------|
| Canonical | `reports/daily/` 固定檔名或 `reports/daily/{strategy_id}/` | `etf-daily/daily_brief.md`, `regime/daily_brief.md` |
| Research | `reports/research/` | 廣度 HTML、RRG 軌跡、copytrade 矩陣 · slot backtest JSON |
